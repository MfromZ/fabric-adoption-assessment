# Data Culture Research

Source: https://learn.microsoft.com/en-us/power-bi/guidance/fabric-adoption-roadmap-data-culture

## Definition
Data culture = behaviors and norms that encourage informed, data-driven decision-making across the organization.

## Three Key Aspects

### 1. Data Discovery
- Ability to locate relevant data assets across the organization
- Different from "search" — discovery reveals metadata even without access
- Tools: OneLake catalog, endorsements (certified/promoted), Microsoft Purview

**Maturity Levels:**
| Level | State |
|-------|-------|
| 100 | Data fragmented, no structures to find it |
| 200 | Scattered efforts to organize, occasional endorsements, data siloed |
| 300 | Central repository (OneLake catalog) in use, endorsement process, basic documentation |
| 400 | Structured consistent processes, data silos are exception, comprehensive data dictionaries |
| 500 | Systematically organized with full lineage, cataloging tools like Purview in use |

### 2. Data Democratization
- Putting data into hands of more users who solve business problems
- Not about removing security — about enabling with proper governance

**Maturity Levels:**
| Level | State |
|-------|-------|
| 100 | Data limited to small roles, gatekeeping, chaotic self-service, enterprise team can't keep up |
| 200 | Limited efforts to expand access, some teams successful |
| 300 | Many people have access, self-service practices spreading |
| 400 | Healthy partnerships, clear accountability, users recognized/rewarded |
| 500 | Governance gives confidence, automated processes for data access |

### 3. Data Literacy
- Ability to interpret, create, and communicate with data accurately

**Maturity Levels:**
| Level | State |
|-------|-------|
| 100 | Decisions on intuition, low confidence with data, preference for static tables |
| 200 | Some teams use data inconsistently, some resistance to data challenges |
| 300 | Majority understand and use data for decisions, visualizations more accepted |
| 400 | Data literacy recognized as skill, training programs, best practices documented |
| 500 | Data literacy is strategic skill, decision intelligence in use, best practices essential |

### Overall Data Culture Maturity
| Level | State |
|-------|-------|
| 100 | Enterprise teams can't keep up, chaotic self-service, no oversight |
| 200 | Multiple teams successful, finding balance between enterprise/self-service |
| 300 | Specific goals for data culture, learnings shared, practices replicated |
| 400 | Goals aligned with org objectives, healthy partnership, individuals recognized |
| 500 | Business value regularly measured with KPIs/OKRs, feedback loops, continuous improvement |
