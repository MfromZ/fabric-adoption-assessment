# User Support Research

Source: https://learn.microsoft.com/en-us/power-bi/guidance/fabric-adoption-roadmap-user-support

## Support Layers
1. **Intra-team support**: Colleagues helping colleagues within a team
2. **Internal community support**: Discussion channels, Q&A forums
3. **Help desk support**: Formal IT help desk (ticketed)
4. **COE support**: Escalated issues, complex problems
5. **External support**: Microsoft support, partners, community

## Maturity Levels
| Level | State |
|-------|-------|
| 100 | Users support themselves or rely on colleagues informally, no structured support |
| 200 | Some support mechanisms emerging, help desk handles basic issues |
| 300 | Multiple layers of support defined, escalation paths clear |
| 400 | All support layers functioning well, user satisfaction tracked |
| 500 | Proactive support, knowledge base well-maintained, continuous improvement |
